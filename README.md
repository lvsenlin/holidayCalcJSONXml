holidayCalcJSONXml - 节假日计算、JSON、Xml三连
============


使用说明
----------------

``` sh
0 工作日 
1 周末 
2 节假日 
```

直接使用
----------------
包含两年的数据

``` sh
holidayJSONAll.json
holidayXmlAll.xml 
```

计算某年节假日
----------------
可在浏览器控制台执行

``` sh
concatHolidayArrAll.js
```

合并两年的节假日
----------------
可在浏览器控制台执行

``` sh
concatHolidayArrAll.js 
```
